### Print array as numbered list
In a new SvelteKit route, print the following array as a numbered list using `{#each}`:

```js
const todoList = [
  'Get out of bed',
  'Make coffee',
  'Read feeds',
  'Have shower',
  'Brush teeth',
  'Get dressed',
  'Leave for work',
  'Get breakfast to go'
]
```

Your output should look like this:

---

**Morning Todos**
1. Get out of bed
2. Make coffee
3. Read feeds
4. Have shower
5. Brush teeth
6. Get dressed
7. Leave for work
8. Get breakfast to go
